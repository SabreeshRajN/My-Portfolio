2. **About** - Personal information and statistics

5. **Projects** - Portfolio of completed projects
6. **Contact** - Contact form and social links

## Technologies Used

- React.js 18.2.0
- CSS3
- React Icons
- Modern JavaScript (ES6+)

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Customization

To customize the portfolio with your information:

1. Update the content in each component file in `src/components/`
2. Replace `src/pic.jpg` with your profile photo
3. Update contact information in `Contact.js`
4. Modify skills, projects, and experience data in respective component files
5. Adjust colors in CSS files to match your preference
